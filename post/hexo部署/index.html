Welcome to Hexo! This is your very first post. Check documentation for more info.

快速开始

安装hexo 

npm install -g hexo-cli

进阶安装和使用 

npm install hexo

安装以后，可以使用以下方式执行 Hexo

npx hexo <command>

新建网站 

npx hexo init [folder]

新建一个网站。 如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

创建新文章 

npx hexo new "My New Post"

More

启动本地服务 

npx hexo server

More

生成静态文件 

npx hexo generate

More

部署 

npx hexo deploy

More

清除缓存文件 (db.json) 和已生成的静态文件 (public)。 

npx hexo clean

脚本部署(运行 sh deploy.sh) 

npx hexo clean

npx hexo generate

npx hexo deploy

ping不通外网的解决方案(centos)

发表于2023-02-05|更新于2024-09-06|网络
|字数总计:149|阅读时长:1分钟|阅读量:

ping 不通外网，绝大部分原因是由于dns服务器没有配置导致的问题

1、修改配置文件添加DNS

#xxx为对应网卡
vi /etc/sysconfig/network-scripts/ifcfg-xxx

#添加DNS
DNS1=114.114.114.114
DNS2=8.8.8.8

2、添加DNS服务器

vi /etc/resolv.conf

#添加如下两行：
nameserver 8.8.8.8
nameserver 8.8.4.4

3、重启网络

sudo nmcli c reload

#或
systemctl restart network

#或
service network restart

4、如果上述操作还是没有成功

#添加默认网关
sudo route add default gw 192.168.1.1

#然后
sudo nmcli c reload

到此一般都没问题了😀
